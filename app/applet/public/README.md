# Video Upload Guide

The Premium Car Wash website is designed to automatically and seamlessly play your 3 custom uploaded videos as a background video showcase loop on page load.

To get them to play instantly:
1. Please upload or drag-and-drop your 3 video files in the **Code Editor File Explorer** (the sidebar panel on the left/top of your AI Studio environment) inside this `public/` folder.
2. Rename the 3 files exactly to:
   - `car-video1.mp4` (your first video, e.g. the Volkswagen Touareg garage interior)
   - `car-video2.mp4` (your second video, e.g. the Ford Ranger Raptor front)
   - `car-video3.mp4` (your third video, e.g. the Lexus RX paint wash)

Once uploaded into `public/`, the website will instantly detect them and loop them automatically in high-definition without needing any manual file loading!
